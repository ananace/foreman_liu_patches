# Foreman LiU Patches

Patches for the Linköping University Foreman install, as a plugin

## Installation

See the [Plugins install instructions, advanced installation from gems](https://theforeman.org/plugins/#2.3AdvancedInstallationfromGems) for information on how to install this plugins.

## Contributing

Bug reports and pull requests are welcome on the LiU GitLab at https://gitlab.liu.se/ITI/foreman_liu_patches or on GitHub at https://github.com/ananace/foreman_liu_patches
